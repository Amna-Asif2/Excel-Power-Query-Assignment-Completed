# Excel-Power-Query-Assignment-Completed – Data Cleaning Practice 🌟

This project demonstrates hands-on practice in **data cleaning and preparation** using an AI-generated dataset, completed as part of my **Hadi E-Learning** course. It showcases techniques to transform messy data into a clean, structured, and analysis-ready format.

---

## 💡 Project Overview

Data analysis starts with **clean and structured data**. This project focuses on mastering the essential steps of **data cleaning** and preparing datasets for meaningful insights.  

**Key tasks performed:**

- Cleaned and standardized messy data containing missing, inconsistent, or incorrect values.  
- Verified calculations for **quantities, unit prices, and total prices**.  
- Organized **sales types, order statuses, and branch/staff information** for consistency.  
- Handled “no sale” and zero-quantity entries to ensure data accuracy.  

---

## ✅ Outcome

- Transformed a **raw dataset** into a **clean, reliable, and analysis-ready dataset**.  
- Enabled meaningful insights into **sales trends, branch performance, and staff efficiency**.  
- Gained **hands-on experience in data cleaning, validation, and preparation**, critical skills in any data analytics workflow.
